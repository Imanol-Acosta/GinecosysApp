# GinecoSysApp

## App de gestion clinica ginecologica

GinecoSysApp es una solución móvil ágil diseñada para la gestión esencial de consultas ginecológicas. Como versión Lite del ecosistema GinecoSys, esta aplicación prioriza la fluidez y el acceso rápido a la información clínica desde dispositivos Android.

Características principales:

Registro de Pacientes: Gestión simplificada de fichas personales.

Control de Consultas: Historial básico de visitas y motivos de consulta.

Interfaz Optimizada: Diseño ligero enfocado en la usabilidad y bajo consumo de recursos.

Arquitectura: Desarrollada nativamente para Android, garantizando estabilidad y rendimiento.
